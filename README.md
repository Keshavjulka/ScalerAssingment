# 📋 Kanban Board

A simple Trello-like project management app built with React and Express.

## 🛠️ Tech Stack

- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** In-memory storage
- **Styling:** CSS

## ✨ Features

- Create, edit, delete lists
- Add, edit, delete cards
- Move cards between lists

## 🚀 Setup & Run

### Backend
```bash
cd backend
npm install
npm start
```
Runs on `http://localhost:5001`

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Runs on `http://localhost:3000`

## 📦 Deployment

- **Backend:** Render.com
- **Frontend:** Vercel.com
- **Repo:** GitHub
